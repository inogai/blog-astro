## Markdown Processing

### [astro](https://astro.build/)

Super cool integration with markdown out of the box.

### [remark](https://github.com/remarkjs/remark)

Markdown processor for the blog.

### [remark-callout](https://github.com/landakram/remark-wiki-link)

Adds callout support to markdown.

### [remark-wiki-link](https://github.com/landakram/remark-wiki-link)

Enables WikiLinks syntax in markdown.

### [Prettier fork for Markdown CJK](https://github.com/inogai/prett)

Custom fork for better CJK text handling in markdown.

### [remark plugin for CJK line wrap](https://github.com/inogai/blog-astro/blob/main/src/lib/remarkCjkWrap.ts)

Custom plugin for proper CJK line wrapping in markdown.
