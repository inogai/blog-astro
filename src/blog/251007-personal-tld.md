在 [Spaceship.com](https://www.spaceship.com/) 購買了域名 <https://inogai.com>。
續費 HKD 80 一年。選擇 Spaceship 的主要理由是發現他支援用 Tap & Go 的 Mastercard
付款。據我所知，許多網絡服務商都不支援 Tap & Go 當作 Credit Card 支付。

Spaceship 的性價比當然比不過內地的域名註冊商，但相對於其他的域名註冊商來說還是相
當不錯的。比如 Namecheap 上的 .com 域名就要到 USD 14.98 / 年。與此同時，內地的域
名註冊商又有一些限制，比如需要實名認證。相較之下，spaceship 的服務還是相當適合我
的。

## 域名的那些坑

第一年售價很便宜，續費價格就高了。這是域名註冊商常見的套路。當然，如果你不打算長
期使用這個域名，那也無妨。不過要是像我這樣，準備用來開設個人網站的話，還是希望可
以長期使用同一個域名的。這樣的話，續費價格就很重要了。

與此同時，由於域名註冊是長期的服務，所以選擇一個可靠的域名註冊商也很重要。一些域
名註冊商的名聲就不是太好。這裏我們不做推薦，大家可以自行搜索相關的評價。

## 非常便宜的域名

在 spaceship 上面，`六位數字.xyz` 的域名續費價格只要 HKD 5 一年，屬於是非常便宜
實用了。如果你的域名不需要向外宣傳，這種域名其實也不錯。畢竟，域名的主要作用是讓
人記住，如果不打算讓別人登錄，或者只是開個遊戲服務器，自己記得住就行了。當然，如
果用於建站，這樣的域名還是顯得比較 sus，網站的 SEO 也不會好。

## TLD 有甚麼用呢？

很多。主要來説，都是與網絡服務有關。比如説我們最常見的網站，
`https://www.google.com` 中的 `google.com` 就是 TLD。而 `www` 則是子域名。當你擁
有了一個頂級域名，你就可以創建無數個子域名，這些子域名可以指向不同的服務器或者不
同的服務。

如果你只是想要用來建一個靜態的部落格網站，那麼，使用 Github Pages 或者 Vercel 這
樣的平台就足夠了。這些平台通常會提供免費的二級域名，比如 `\*.github.io`。也有一
些專門分發二級域名的服務，比如 `eu.org`。不過，如果你想要更專業一點，或者想要更
高的自由度，那麼擁有一個頂級域名就很有必要了。

目前我的 TLD `inogai.com` 部署了兩個服務：

1. `www.inogai.com`: 指向 Github Pages 的靜態網站，是我的個人 Portfolio。
2. `wakapi.inogai.com`: 指向 Spaceship VPS 上的
   [Nginx Proxy Manager](https://nginxproxymanager.com/)，把流量反代理到一個自建
   的 [Wakapi](https://github.com/muety/wakapi) 服務，用來統計自己的 Coding 活動
   。

一個 domain 只能指向一個 IP 地址，不能指定端口。不過，我們可以通過一個 Routing
的服務，讀取用户請求的 domain，然後把請求轉發到不同的服務器或者不同的端口。從而
實現多個服務共用一個 IP 地址的效果。這也是爲什麼我會在 VPS 上面部署 Nginx Proxy
Manager 的原因。

## 電郵

除此之外，擁有一個 TLD 還可以用來創建自定義的電子郵件地址，比如
<****@inogai.com>，這是我目前正在考慮的事情。不過，這需要額外的郵件服務。你也可
以把這個郵件地址轉發到你的個人郵箱，比如 Gmail。

## 結語

關於域名的事情，差不多就説到這邊。建站的具體細節，我會在後續的文章中再做介紹。總
的來説，擁有一個自己的域名果然是一種程序員的浪漫。
