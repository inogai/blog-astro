Companion app for the 3300 Smart Curtain project, enabling bluetooth remote control of curtain motors.

## Features

- **Bluetooth Connectivity**: Direct connection to smart curtain devices
- **Remote Control**: Open, close, and adjust curtains from your phone
- **Real-time Status**: Monitor curtain position and device status
- **Modern UI**: Clean, responsive interface built with React Native

## Tech Stack

- **Frontend**: React Native with TypeScript
- **Native Modules**: Kotlin for Android bluetooth integration
- **Styling**: Tailwind CSS via NativeWind
- **Build**: Expo CLI for cross-platform development
