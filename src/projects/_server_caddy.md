My self-hosted server configuration using Caddy as a reverse proxy with automatic HTTPS, Docker containerization, and SSL certificate management.

## Features

- **Caddy**: Modern web server with automatic HTTPS
- **Reverse Proxy**: Multiple services behind a single domain
- **Docker**: Containerized services for easy deployment
- **Automatic SSL**: Let's Encrypt integration via Caddy
- **Security**: Configured with best practices for self-hosting

## Tech Stack

- **Web Server**: Caddy v2 with automatic HTTPS
- **Containerization**: Docker & Docker Compose
- **OS**: Linux server
- **SSL**: Let's Encrypt via Caddy
