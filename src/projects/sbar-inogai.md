My personal Sketchybar configuration written in Lua, designed for AeroSpaceWM window manager. Available as a Nix flake.

## Features

- **SketchyBar**: Powerful macOS bar customization
- **Lua Config**: Lightweight and flexible configuration language
- **AeroSpaceWM**: Seamless integration with AeroSpace tiling window manager
- **Nix Flake**: Reproducible installation via Nix

## Tech Stack

- **Bar Engine**: SketchyBar for macOS status bar
- **Language**: Lua for configuration logic
- **Window Manager**: AeroSpaceWM integration
- **Package Manager**: Nix flakes for distribution

## Components

- **assets**: Icons and images
- **config**: Bar configuration files
- **agents**: Background agents for system monitoring
