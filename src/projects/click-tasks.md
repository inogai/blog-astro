A mobile app that combines task management with expense tracking, enhanced with AI voice recognition.

## Features

- **Task Management**: Create and manage todo items with voice input
- **Expense Tracking**: Log and track personal expenses
- **LLM Integration**: AI-powered task suggestions and expense categorization
- **Voice Commands**: Use voice to add tasks and expenses
- **Cross-platform**: Built with React Native for iOS and Android

## Tech Stack

- **Frontend**: React Native with TypeScript
- **Voice Recognition**: react-native-voice for speech-to-text
- **Styling**: Tailwind CSS via NativeWind
- **Package Manager**: pnpm
