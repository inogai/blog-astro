Real-time ETA tracking for Hong Kong's public transportation system.

## Features

- **Multi-transport Support**: LRT, buses, and more
- **Real-time Updates**: Live arrival predictions
- **Modern UI**: Clean interface with shadcn-vue components
- **Mobile-friendly**: Optimized for on-the-go usage

## Tech Stack

- **Framework**: Vue 3 with TypeScript
- **Components**: shadcn-vue for UI primitives
- **Styling**: UnoCSS for utility-first styling
- **Build**: Vite for fast development and production builds
